# Taldau

Переводчик офисных документов (.pptx, .docx, .xlsx) с веб-интерфейсом. Поддерживает 13 языков, параллельную обработку файлов и кэширование переводов.

## Возможности

- Перевод документов PowerPoint, Word и Excel с сохранением форматирования
- 13 языков: русский, казахский, английский, турецкий, узбекский, кыргызский, китайский, арабский, немецкий, французский, испанский, корейский, японский
- Параллельная обработка XML-файлов внутри документа (по числу ядер CPU)
- SQLite-кэш переводов для ускорения повторных запросов
- Drag & drop загрузка файлов
- Прогресс-бар перевода в реальном времени
- Просмотр логов с фильтрацией по уровню
- Список ранее переведённых файлов с возможностью скачивания

## Стек

- [Bun](https://bun.sh) — рантайм, HTTP-сервер, сборщик
- [HTMX](https://htmx.org) + [Alpine.js](https://alpinejs.dev) — интерфейс без сборки фронтенда
- [bun:sqlite](https://bun.sh/docs/api/sqlite) — кэш переводов
- [JSZip](https://stuk.github.io/jszip/) — работа с Office-документами (OOXML)
- Google Translate API — движок перевода

## Установка

```bash
bun install
```

## Запуск

```bash
bun run start
```

Сервер запустится на `http://localhost:3333`.

Для разработки с автоперезагрузкой:

```bash
bun run dev
```

## Переменные окружения

| Переменная | По умолчанию | Описание |
|------------|-------------|----------|
| `PORT` | `3333` | Порт HTTP-сервера |

## Структура проекта

```
index.ts            — HTTP-сервер, маршруты, обработка документов
public/index.html   — веб-интерфейс (HTMX + Alpine.js)
src/
  parsers.ts        — XML-парсеры для PPTX/DOCX/XLSX
  translate.ts      — обращение к Google Translate, батчинг
  db.ts             — SQLite-кэш переводов
  logger.ts         — логирование в файл и в память (для UI)
original/           — загруженные оригиналы
result/             — переведённые документы
logs/               — файлы логов по дням
translations.db     — база кэша переводов
```

## API

| Метод | Путь | Описание |
|-------|------|----------|
| `GET` | `/` | Веб-интерфейс |
| `POST` | `/api/translate` | Загрузка и перевод документа (multipart/form-data) |
| `GET` | `/api/progress/:id` | Прогресс задачи перевода (HTML) |
| `GET` | `/api/download/:file` | Скачивание переведённого документа |
| `GET` | `/api/results` | Список переведённых файлов (HTML) |
| `GET` | `/api/languages` | Список поддерживаемых языков (JSON) |
| `GET` | `/api/cache` | Статистика кэша (HTML) |
| `DELETE` | `/api/cache` | Очистка кэша |
| `GET` | `/api/logs` | Последние записи логов (HTML) |

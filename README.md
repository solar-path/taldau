# Taldau

Переводчик офисных документов (.pptx, .docx, .xlsx) с веб-интерфейсом. Поддерживает 13 языков, параллельную обработку файлов и кэширование переводов.

## Возможности

- Перевод документов PowerPoint, Word и Excel с сохранением форматирования
- 13 языков: русский, казахский, английский, турецкий, узбекский, кыргызский, китайский, арабский, немецкий, французский, испанский, корейский, японский
- Параллельная обработка файлов
- Кэширование переводов
- Drag & drop загрузка
- Прогресс-бар в реальном времени
- Логирование с фильтрацией по уровню
- Список переведённых файлов с возможностью скачивания

## Стек

- [Bun](https://bun.sh) — рантайм, HTTP-сервер
- [HTMX](https://htmx.org) + [Alpine.js](https://alpinejs.dev) — интерфейс
- SQLite — кэш
- [JSZip](https://stuk.github.io/jszip/) — работа с документами

## Установка

```bash
bun install
```

## Запуск

```bash
bun run start
```

Сервер запустится на `http://localhost:3333`.

Для разработки с автоперезагрузкой:

```bash
bun run dev
```

## Переменные окружения

| Переменная | По умолчанию | Описание |
|------------|-------------|----------|
| `PORT` | `3333` | Порт HTTP-сервера |

## Структура проекта

```
index.ts            — HTTP-сервер
public/index.html   — веб-интерфейс
src/
  parsers.ts        — парсеры документов
  translate.ts      — сервис перевода
  db.ts             — кэш
  logger.ts         — логирование
original/           — загруженные оригиналы
result/             — переведённые документы
logs/               — логи
```
